# Agar.io Clone

Упрощенная клон-версия популярной игры Agar.io, реализованная на Python с использованием библиотеки Pygame.

## Описание игры

Игрок управляет клеткой, которая должна поглощать еду и другие клетки меньшего размера. Цель игры - набрать 700 очков, поедая еду и врагов.

## Особенности

- **Движение клетки** - управление курсором мыши
- **Система роста** - клетка увеличивается при поедании еды и врагов
- **Искусственный интеллект** - враги перемещаются по карте с простым ИИ
- **Система очков** - подсчет очков за поедание еды и врагов
- **Условия завершения**:
  - Победа: достижение 700 очков
  - Поражение: поглощение более крупным врагом
